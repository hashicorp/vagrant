# Validate

**Command: `vagrant validate`**

This command validates your [Vagrantfile](/vagrant/docs/vagrantfile/).

## Options

- `--ignore-provider` - Ignores provider config options.

## Examples

```shell-session
$ vagrant validate
Vagrantfile validated successfully.
```
